# Neo4j-Cypher
Folder to hold scripts, notes and other course resources for Neo4j Udemy course.

## Basic Cypher Commands

| Symbol | Meaning|
| --- | --- |
| () | Node |
| -- | Relationship e.g. ()--() |
| --> | Relationship direction |
| -[]- | Relationship variable:type |
| {} | node properties |
